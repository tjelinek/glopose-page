Dense Matchers for Dense Object Reconstruction

Abstract

We present a zero-shot, image-based framework for object representation appropriate for pose estimation, tracking, or self-relocalization. Our method assumes an RGB input video capturing the object, with an annotation of the object's pose in the first frame together with a segmentation mask. It selects a minimalistic image-based representation of the object from the onboarding video frames in an online fashion, based on a matching score. Subsequently, we use a Structure-from-Motion pipeline to recover the semi-dense point cloud and metric camera poses aligned with the reference image pose. We demonstrate that the representation is sufficient for various tasks, such as unseen object pose estimation, 6DoF pose tracking, or self-relocalization.
